Introduction

jthesis is a LaTeX 2e class file intended for formatting a PhD thesis in the Department of Electrical Engineering at the University if Edinburgh. You should not need to hack around with \vspace and other such commands when using this class file.

Bugs and feedback

Please report all problems and suggestions to the maintainer above. If you find you need to hack this style file to make it do something differently to suit your taste please consider adding it as an option, or contact the maintainer to have it added as an option. This will avoid many partially-working and undocumented variants circulating around.

Requirements

LaTeX 2e (1996/12/01) or later. The following packages are needed: report, graphicx, fancyheadings, citesort, doublespace, float, ifthen, caption, afterpage.

Compiling the example

First copy the whole of the ThesisStyleFile directory into your own account: 
cp -r /home/sasg/Latex/ThesisStyleFile  MyCopy
Now run LaTeX and BibTeX and view the result: 
cd MyCopy
texsetup
latex thesis
bibtex thesis
latex thesis
latex thesis
dvips thesis.dvi -o thesis.ps
gv thesis.ps

List of options, commands and features

Most of the new features of this style file that you need to know about are documented in comments in the example thesis files, however a list of new commands and things performed by the style file is given below.
 

Document options

nohyphens
This document option turns off hyphenation. The University guidelines suggest this but it can make for some ugly formatting. It also allows the formatting to be more sloppy so that the chance of long lines
sticking into the right hand margin is reduced. Try it and see if you like it or not.
doublespace
Normally you want this option to give one-and-a-half spacing, however for printing out draft copies during writing removing this option will save paper.
twoside
Produce double sided output.

General style and appearance

  • 11pt type is used, on A4 paper, single column
  • times font family is used (easier to read and gives a much smaller postscript file)
  • margins are set according to University guidelines
  • headings reflect the current chapter title
  • chapter title style is redefined, with nice horizontal ruling!
  • paragraph indentation and spacing is redefined, as is spacing after section titles
  • double sided output (twoside) works, chapters always start on right hand page
  • quotations are single spaced (as defined by University guidelines)
  • bibliography is retitled 'references'
  • roman page numbering at start, arabic later for real chapters
  • double spacing is not used for contents, lists of tables and figures and references

    • Tips

  • use commands from the graphicx package for inclusion of graphics
  • use [H] as a float position specifier to mean really-here
  • use amstex package for better maths
  • use subfigure package for multiple parts to figures with subcaptions

    • Commands

    \crestfile{filename}
    Specify the name of the EPS file to use for University Crest on the front title page.
    \minorchapter{title}
    Start a new minor chapter. Minor chapters are the ones which come before your first real chapter, for example acknowledgements.
    \startchapters
    Indicate that real chapters are about to start. This currently just sets arabic page numbering, but may do more in the future.
    \contentsandlists
    Insert a table of contents, list of figures and list of tables (with table of contents entries).
    \insertreferences{file1,file2,file3}
    Insert references using IEEE style and single spacing.
    \image[options-to-\includegraphics]{filename}{label}{caption}
    Useful macro to insert a centered float  using the graphicsx package.
     \flushfloats
    The current page will be finished as normal, the next new page will contain any floats
     which have not yet been placed (if any). This is useful if a few large or full-page floats
    are pushing other smaller diagrams towards the end of your chapter.
